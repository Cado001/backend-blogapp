## Backend Repo

This is the backend for a full stack blog app. It is built using Express and MongoDB.

## Build Instructions
1. git clone https://github.com/Cado001/frontend-blogapp
cd frontend-blogapp
3. obtain .env for environmental variables
4. npm i 
5. node .
